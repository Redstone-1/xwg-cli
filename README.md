# 用前准备

node 版本 14.18+ 或 16+ 或更高

`npm (yarn、pnpm) i (add) xwg-cli -g`

`npx xwg-cli create <project-name>`

# 查看命令详情

`xwg` | `xwg --help` | `xwg -h`

# 查看版本

`xwg --version` | `xwg -V`

# 新建项目

`xwg create <project-name> -f (--force)` 

例如：`xwg create vue-template -f`
# 查看命令详情

`xwg <command> --help`

例如：`xwg create --help`
